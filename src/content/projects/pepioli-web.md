## O projektu

Tento web jsem vytvořila jako svůj osobní prostor pro sdílení blogových příspěvků, receptů a projektů.

## Technologie

- **Astro** - Moderní framework pro statické weby
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Typově bezpečný JavaScript
- **Decap CMS** - Git-based CMS pro správu obsahu

## Funkce

- Responzivní design optimalizovaný pro mobily
- Rychlé načítání díky statickému generování
- Přístupný design s ohledem na a11y
- Jednoduchá správa obsahu přes CMS
